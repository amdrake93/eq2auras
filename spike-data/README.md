# spike-data

Drop-off folder for ferrying spike diagnostic logs from the Windows/ACT box to the Mac for analysis.

- Put the newest `spike-*.jsonl` (from `%APPDATA%\Advanced Combat Tracker\eq2auras\logs\`) here — upload via GitHub web, or `git add` (the repo's `.gitignore` has an exception for `spike-data/*.jsonl`).
- These are **transient analysis artifacts** — removed from tracking once analyzed.
